# 🇸🇦 النسخة العربية

## 🚀 رؤية المشروع

هذا المشروع ليس فقط لإدارة الجداول الدراسية، بل هو **مساعد طلابي ذكي** يهدف إلى مساعدة الطالب في حياته الأكاديمية من خلال واجهة محادثة بسيطة وذكية.

يساعد النظام الطالب في:

* 📅 إدارة الجدول الدراسي
* 🔔 استقبال إشعارات وتذكيرات
* 📚 متابعة الخطة الدراسية
* 🤖 الحصول على اقتراحات للمواد القادمة
* 📤 رفع الواجبات
* 📖 فهم الأنظمة واللوائح الجامعية

## ⚙️ طريقة التشغيل

### 1) استنساخ المشروع

```bash
git clone https://github.com/Yzeeyd/Chat-Assistant-for-Students.git
cd Student_bot
```

### 2) إنشاء بيئة افتراضية

```bash
python -m venv .venv
```

### 3) تفعيل البيئة

```bash
.venv\Scripts\activate
```

### 4) تثبيت المكتبات

```bash
pip install -r requirements.txt
```

### 5) إعداد ملف البيئة

انسخ `.env.example` وغيّر اسمه إلى `.env` ثم ضع القيم المناسبة.

### 6) تشغيل المشروع

```bash
python -m uvicorn app.main:app --reload
```
