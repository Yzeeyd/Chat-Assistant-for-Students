# نظام المساعد الطلابي — نسخة داخل نفس المشروع

هذه النسخة تجمع كل شيء داخل **نفس المشروع**:
- Backend بـ FastAPI
- واجهة ويب يقدّمها FastAPI مباشرة
- تسجيل دخول JWT
- Agentic AI مبني على أدوات
- قاعدة بيانات SQLite افتراضيًا مع إمكانية التحويل لاحقًا

## الموجود في النسخة
- إنشاء حساب وتسجيل دخول
- شاشة Dashboard
- شاشة Chat
- أدوات الجدول
- أدوات التذكيرات
- أدوات الخطة الأكاديمية
- البحث في الأنظمة الجامعية
- عرض صور القاعات من مجلد `uploads/rooms`

## لماذا هذه النسخة مناسبة الآن
لأنك طلبت أن تكون الواجهة **داخل نفس المشروع مبدئيًا**، وهذه أفضل نقطة بداية من ناحية:
- بساطة التشغيل
- سهولة العرض على الدكتور
- تقليل مشاكل الربط بين frontend و backend
- قابلية تطوير المشروع لاحقًا بدون إعادة بناء كاملة

## التشغيل
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
copy .env.example .env
python -m uvicorn app.main:app --reload
```

ثم افتح:
- التطبيق: `http://127.0.0.1:8000/`
- التوثيق: `http://127.0.0.1:8000/docs`

## ملاحظات
- أضف `OPENAI_API_KEY` داخل `.env`
- القاعدة الافتراضية الآن SQLite حتى يشتغل المشروع بسهولة
- تقدر لاحقًا تنقل إلى MySQL بدون تغيير شكل المشروع
